# Tauri Screen Capture Example

This example uses plain HTML, CSS, and JavaScript with Vite to exercise the screen capture plugin.

On macOS 14+, enabling the `macos-screencapturekit` feature exposes the native annotation toolbar. Pen, eraser, undo, and clear are composited into the shared video; turning annotation mode off restores click-through to the shared window.

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Tauri](https://marketplace.visualstudio.com/items?itemName=tauri-apps.tauri-vscode) + [rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer).
